## Spis treści
- Sprawozdanie nr 1 - Struktury danych: tablice i drzewa
- Sprawozdanie nr 2 - Algorytm Dijkstry 
- Sprawozdanie nr 3 - Algorytm Bellmana-Forda
- Sprawozdanie nr 4 - Sortowanie bąbelkowe (Bubble Sort)
- Sprawozdanie nr 5 - Sortowanie szybkie (QuickSort)
